# Palettes

`colour_palette_name.csv`

## Columns

- `name`: (`optional`) colour name
- `color`: color hex code i.e. `#RRGGBB`
- `type`: (`optional`)
  - `background`
  - `text`
  - `accent`: main colours in chart
    - `accent_ui`: special colours in chart/ui for example: safe vs danger
    - `accent_s`: secondary accent
- `note`: (`optional`)
